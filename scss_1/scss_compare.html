整體而言，SASS 以及 SCSS 主要差異在於一個利用縮排的方式撰寫，一個利用括號以及分號進行處理，而 Stylus 涵蓋範圍比較大，可以吃的形式也比較多元。
LESS 是由 Alexis Sellier 所設計的工具，原本是用 Ruby 實作，用來將 LESS 語法轉換成 CSS 語言。SASS (Syntactically Awesome Stylesheets) 是由 Hampton Catlin 設計，然後由 Natalie Weizenbaum 實作的一種樣式語言。LESS 與 JavaScript 一樣，有用 {...} 的大括號框起來，而 SASS 則像 Python 一樣，沒有用大括號，而是透過換行與縮排來表現這種層次結構。SASS 另一個變種是加上大括號的版本，稱為 SCSS，這兩種風格可以互相轉換，可依照習慣挑選使用。
